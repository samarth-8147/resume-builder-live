# Resume Builder Live

## Overview

Resume Builder Live is a web-based application that helps users create professional resumes quickly and easily. Users can enter their personal details, education, skills, work experience, and projects, then generate a well-formatted resume instantly.

## Features

* User-friendly interface
* Create and edit resume details
* Add personal information, education, skills, and experience
* Real-time resume preview
* Download or print resume
* Responsive design for desktop and mobile devices
* Easy customization

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap (if used)

## Installation

1. Clone the repository:

git clone https://github.com/samarth-8147/RESUME-BUILDER-LIVE.git

2. Open the project folder:

cd RESUME-BUILDER-LIVE

3. Open `index.html` in your browser.

## Usage

1. Enter your personal information.
2. Add educational qualifications.
3. Include skills, projects, and work experience.
4. Preview your resume in real time.
5. Download or print the generated resume.
